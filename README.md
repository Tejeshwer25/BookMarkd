# 📚 BookMarkd

> *A thoughtful space for readers to capture, reflect, and revisit what they read.*

BookMarkd is a modern iOS application designed as a **personal reading companion**.  
It goes beyond tracking books by helping users **capture meaningful quotes, reflect on their reading journey, and discover what to read next using AI**.

Built with a strong focus on **editorial design, clean architecture, and user intent**, BookMarkd blends the feel of a **private journal** with the power of a **smart recommendation system**.

# ✨ Core Idea

Most reading apps focus on *what* you read.  
BookMarkd focuses on:

> **What stayed with you after reading.**

# 🚀 Features

## 📚 Library Management
- Organize books into:
  - Currently Reading
  - Finished
  - Wishlist
- Add books via:
  - Search
  - Manual entry
  - AI recommendations

---

## ✍️ Quotes & Notes
- Save meaningful quotes from books
- Attach notes and reflections
- Access all your highlights in one place

> No need to carry your book to revisit your favorite lines.

---

## 🖼️ Quote Sharing
- Export beautifully styled quote cards
- Multiple visual themes (editorial, minimal, expressive)
- Share across social platforms

---

## 🤖 AI-Powered Recommendations
- Personalized book suggestions using **Foundation Models**
- Based on:
  - Preferred genres
  - Reading activity
- One-tap add to wishlist

---

## 📊 Reading Insights
- Track your reading journey with:
  - Books finished
  - Average rating
  - Quotes saved
- Visualize reading activity over time

---

## 🎯 Genre Personalization
- Select preferred genres
- Drives smarter recommendations

---

## ⚡ Performance Enhancements
- Book cover image caching
- Smooth UI interactions
- Optimized data handling with SwiftData

---

# 🧱 Tech Stack
